# Predicting Patient Medication Adherence Using Machine Learning

This project applies supervised machine learning classification models to predict whether patients will adhere to prescribed medication regimens.

## Project Overview

Medication adherence is a major healthcare challenge associated with increased hospitalization rates, disease progression, and healthcare costs. This project uses machine learning techniques to identify patterns associated with medication non-adherence.

## Models Used

- Decision Tree Classifier
- Logistic Regression

## Dataset

Source:
https://www.kaggle.com/datasets/rabieelkharoua/predicting-medication-adherence-dataset

- 5,000 patient records
- 14 variables
- Binary target variable: Adherence

## Techniques Applied

- Exploratory Data Analysis (EDA)
- Feature Engineering
- One-Hot Encoding
- Label Encoding
- Standardization
- Cross Validation
- ROC Curve Analysis
- Confusion Matrix Evaluation

## Key Findings

- Previous adherence behavior was one of the strongest predictors
- Mental health and insurance coverage significantly influenced adherence
- Logistic Regression outperformed Decision Tree in ROC AUC and overall stability
